# GMO ペパボ株式会社（Pepabo）

## 公司简介

GMO ペパボ株式会社（GMO Pepabo, Inc.）是日本一家以"提供让人开心的服务"为理念的互联网企业，
总部位于福冈，在东京、鹿儿岛设有办公室。隶属 GMO 互联网集团。

公司成立于 2003 年（最初名称为 paperboy&co.），2014 年更名为现在的 GMO ペパボ。
公司在东京证券交易所上市，股票代码 3633。

## 主要服务

Pepabo 运营多个面向 C 端创作者和小企业的互联网服务：

- **ロリポップ！(Lolipop!)**：日本规模最大的共享租用主机服务之一，月活用户超过 200 万
- **ムームードメイン (Muumuu Domain)**：域名注册服务，是日本知名的独立站建站套件
- **カラーミーショップ (Color Me Shop)**：电商网店搭建平台，主打"轻松开店"
- **SUZURI**：印刷 T 恤、马克杯等周边的"创作者经济"平台
- **minne**：手作品 C2C 交易平台，日本同类规模最大

## 技术栈

Pepabo 的技术栈以 Ruby on Rails 为核心，同时大量使用 Go、Python、TypeScript。
基础设施层用 Kubernetes、GCP、AWS。前端框架以 React 和 Vue 为主。

公司有强烈的开源文化，曾贡献过 mruby 的相关项目。

## 招聘与培养

Pepabo 每年招收新卒（应届毕业生）作为 16 期、17 期……持续编号至今。
第 16 期新人（即 2026 年 4 月入职）会经过约 3 个月的研修，
涵盖 Web 基础（HTTP/HTML/CSS）、Ruby/Rails 实战、Python 与 AI、协作流程等。
研修结束后会被分配到产品线团队，比如 minne、SUZURI、Lolipop 等。
